### 关于我

我主要维护 Hackintosh / OpenCore 相关项目，整理不同台式机、迷你主机、笔记本平台的 EFI 配置、硬件适配记录和常用驱动教程。

这里的仓库更像一个可检索的黑苹果资料库：每个项目对应一套具体硬件，尽量写清楚 CPU、显卡、芯片组、macOS 版本、OpenCore 版本和已知适配情况。

### 使用说明

不同机器的硬件批次、BIOS 版本和外设组合可能不同，EFI 请先核对配置再使用。建议在替换 EFI 前备份原文件，并准备可恢复的启动盘。

如果某个仓库帮到了你，可以点一个 Star；如果遇到兼容性问题，也欢迎通过 Issue 补充硬件信息和现象描述。
